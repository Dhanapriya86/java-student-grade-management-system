# Java Student Grade Management System

A simple Student Grade Management System built using Java. This application accepts student details, calculates average marks, and generates grades.

## Features

* Student Name Input
* Marks Entry for 3 Subjects
* Average Calculation
* Grade Generation
* Student Report Display

## Technologies Used

* Java
* Arrays
* Classes and Objects
* Loops and Conditional Statements

## How to Run

1. Compile the program

javac StudentGradeManagement.java

2. Run the program

java StudentGradeManagement

* Arrays
